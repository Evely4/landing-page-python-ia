# 🚀 Landing Page - Python e IA

Landing page desenvolvida com foco em conversão, simulando uma página real de marketing digital para um curso de Python e Inteligência Artificial.

---

## 💡 Objetivo

Criar uma interface web moderna, clara e funcional, com foco em experiência do usuário e incentivo à ação (captura de leads).

---

## 🌐 Acesse o projeto

👉 https://evely4.github.io/landing-page-python-ia/

---

## ⚙️ Tecnologias utilizadas

- HTML
- CSS (Tailwind via CDN)
- JavaScript
- Git e GitHub

---

## 📱 Funcionalidades

- Layout responsivo (desktop e mobile)
- Navegação com scroll suave
- Estrutura completa de landing page:
  - Hero (apresentação)
  - Benefícios
  - Como funciona
  - Prova social
  - Call to Action (CTA)
- Animações ao rolar a página
- Contador dinâmico
- Formulário de captura de lead (simulado)
- Botão com redirecionamento para WhatsApp

---

## 🤖 Uso de IA

Utilizei IA como apoio durante o desenvolvimento para:

- Estruturar a página
- Organizar o layout
- Melhorar o código
- Acelerar o desenvolvimento
